Criando Sites em HTML e CSS

Aprendendo HTML

HTML básico

Compreender os conceitos da linguagem de marcação HTML5. Uso de conceitos e sintaxe fundamentais de como aplicar HTML ao texto, bem como criar hiperlinks e fazer uso do HTML para estruturar uma página da web.

HTML Intermediário

Compreender como organizar seu conteúdo HTML5 usando corretamente as estruturas semânticas da linguagem. Tags Principais e complementares. Integração com outras linguagens como CSS, JavaScript e PHP para melhorar a experiência do usuário e estruturar sites dinâmicos.

HTML Avançado

Manipulação de Tabelas, Frames, Formulários, Cor, sons e âncoras. /p>

Aprendendo CSS

CSS básico

Conceiros iniciais de linguagem de marcação e folhas de estilo, Seletores e sua sintaxe. Estilizando boxes e textos.

CSS Intermediário

Diferentes configurações de tela, métodos de layout tradicionais envolvendo flutuação e posicionamento e novas ferramentas de layout como o flexbox.

CSS Avançado

Uso de rotação, inclinação, escala e translação utilizando páginas de estilo. Ferramentas e metodologias para escrever CSS mais sucinto, sustentável e escalável. Utilização dO recurso @font-face do CSS3 permite que você use fontes personalizadas na web de uma forma acessível e manipulável.
